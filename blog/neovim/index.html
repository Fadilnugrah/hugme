Configurasi untuk neovim + autocomplete python?

October 22, 2021

configurasi neovim#

Hallo kawan kawan, sekarang saya ingin membahas gimana cara memasang configurasi neovim theme + autocomplete python, neovim adalah sebuah code editor yang basednya dari vim khusus command line dan gak kalah ama yg sebelah seperti VSCode, py-charm, sublime text, ternyata neovim/vim juga gak kalah keren loh.

Sekarang kita bahas apa saja yg perlu di install?, ya tentu saja install terlebih dahulu neovim nya

$home: apt-get install neovim -y
$home: apt-get install nodejs-lts
$home: apt-get install git -y

Jika sudah install package di atas mari kita eksekusi plug nya..

$home: git clone https://github.com/Gitcomeon8/dotfiles
$home mv dotfiles/nvim .config/

Nah itu sudah di pindahkan yaa plug serta configurasinya kedalam file .config, Setelah itu masuk ke neovim.

$home: nvim

Nanti akan masuk ke tampilan neovim nya, kalian cukup ketik perintah di bawah ini

:PlugInstall
Kalau sudah selanjutnya install autocompletenya
:CocInstall coc-pyright

Tunggu hingga selesai :)

NOTE! Jika terjadi Error pada coc nya install di bawah ini

$home: npm i https://github.com/Qix-/node-error-ex
$home: npm install

Terimakasih ….
